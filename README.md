# Telegram-Bot-Template

## Overview

This is a Telegram bot template based on .NET 5 and webhook.

## Usage
To run the bot locally on your machine and use a webhook, you should use ngrok, which allows you to generate public tonnel. Also change BotConfiguration.Hook.Url property on ngrok public url and Access Token.
